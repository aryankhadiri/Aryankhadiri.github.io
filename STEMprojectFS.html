STEM Temperature Project
Python Object Oriented Programming

Developed in Python Programming language by OOP with no GUI.

Summary

I developed this application for a programming class I took in foothill college in order to extend my knowledge on Object Oriented Programming and Python. It was my first experience with Python and that was the summer when I fell in love!(With Python). The application reads the csv file and by the menu given to the user, allows it to view different representations of the data. This was not complicated at all by any measure but it was amazingly helpful in terms of learning Python and OOP challenges. Therefore I thought I would mention my focused experience on OOP with python
Link to Github Repository

Features

  • Allows the user to change the units of temperature between Celsius, Fahrenheit and Kelvin. Then all the data represented in the application will be by the selected format.
  • It allows the user to toggle the sensors which then it will effect the statistics since the data collected from those sensors will not be considered.
  • It provides the user with a summary statistics view and a temperature by date and time view
