myInstagram

Part of the CodePath IOS University Course assignment.

Summary

This was an individual practice of IOS UI and Parse Database. myInstagram is trying to be similar to Instagram application which allows you to create user and upload pictures and react to others' post. I developed this application mainly to learn more about connecting a database to IOS and practice the UI and improve my skills in those particular areas. This was meant to help me get ready for the OrderBud Application which is the main project of the class.
Link to Github Repository

Features

  • Users can create a new account and login
  • Users can add photos from their phone library or their camera and post them which then they will be displayed on the homepage
  • Users can comment on other users posts
