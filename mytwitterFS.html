myTwitter

Part of the CodePath IOS University Course assignment.

Summary

This was an individual practice of IOS UI and the connecting to the APIs such as Twitter. myTwitter is meant to mimic the actual Twitter application but only a few features of it. Such as the home view that displays the recent tweets of your fallowing accounts. It also allows you to make a tweet, favorite a tweet and also retweet. In addition, there is a login feature after you attempt to login you are prompted to the Twitter authorizing API which will allow the app to log you in.
Link to Github Repository

Features

  • Users can log in to their twitter account
  • User can view the recent tweets of their following accounts
  • User can post a tweet, favorite a tweet and retweet.
